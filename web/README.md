# Interfaz Web

## Objetivo

La interfaz web permitirá controlar el sistema de riego desde cualquier navegador conectado a la red.

## Funciones

- Ver estado de los nodos.

- Activar y desactivar válvulas.

- Crear programas de riego.

- Consultar historial.

- Ver alarmas.

- Configurar el sistema.

- Mostrar sensores.
