# ESP32 Nodo

## Descripción

El ESP32 Nodo recibe comandos desde el Maestro mediante LoRa y controla las salidas asignadas.

## Funciones

- Recibir comandos LoRa.

- Activar relevadores.

- Desactivar relevadores.

- Enviar confirmaciones (ACK).

- Reportar estado.

- Leer sensores (futuro).

- Reportar alarmas.
