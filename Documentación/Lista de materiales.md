# Lista de Materiales

## Maestro

- ESP32 DevKit V1

- Módulo LoRa SX1276 o SX1278

- Fuente de alimentación 5V

- Caja de protección IP65

- Antena LoRa

- Router Wi-Fi o Starlink

## Nodo

- ESP32 DevKit V1

- Módulo LoRa SX1276 o SX1278

- Módulo relevador (1, 2, 4 u 8 canales según la aplicación)

- Fuente de alimentación

- Caja IP65

- Antena LoRa

## Actuadores

- Electroválvulas de 24 VAC

- Relevadores de aislamiento

- Fusibles de protección

- Borneras

- Conectores impermeables

## Sensores (Opcionales)

- Humedad del suelo

- Temperatura

- Humedad ambiental

- Presión

- Caudal

- Nivel de tanque

## Comunicación

- LoRa 433 MHz o 915 MHz (según la región)

- Confirmación de recepción (ACK)

- Comunicación bidireccional

- Alcance de varios kilómetros con visión despejada
