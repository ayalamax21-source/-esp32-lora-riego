# ESP32 Maestro

## Descripción

El ESP32 Maestro es el controlador principal del sistema de riego inteligente.

Sus funciones son:

- Administrar la comunicación LoRa.

- Conectarse a la red Wi-Fi.

- Mostrar la interfaz web.

- Controlar los nodos remotos.

- Programar horarios de riego.

- Registrar eventos.

- Recibir información de sensores.

- Supervisar el estado de cada nodo.
